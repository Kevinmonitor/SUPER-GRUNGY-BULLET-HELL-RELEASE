--- CONTROLS ---

If you've played Touhou, all of these should be familiar to you.

Arrow Keys: Navigate Menu/Movement
Z: Hold to Fire/Confirm
X: Use Burst Attack (only when Gauge > 100%)/Cancel
Shift: Hold to Slow Movement/Focus
Esc: Pause when In-game

You can use the config.exe to rebind keys.
Please note that the new rebindings will not be reflected in the tutorial text.
